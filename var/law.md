# Underscore Redirects

Add underscore shortlinks here as Markdown reference definitions. The examples
below are escaped so this starter file does not create live redirects; omit the
leading backslash when adding a real redirect.

```md
\[_1]: https://example.com/first-law-link
\[_2]: https://example.com/second-law-link
\[_a]: https://example.com/lettered-law-link
\[_a1]: https://example.com/letter-and-number-law-link
\[_22]: https://example.com/later-numbered-law-link
```

The 404 redirect page reads this file for paths such as `/_1`, `/_2`,
`/_a`, `/_b`, `/_a1`, and `/_44`.
